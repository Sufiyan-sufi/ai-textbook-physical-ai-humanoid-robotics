# Mesh Files for Humanoid Robot

This directory contains 3D mesh files for the humanoid robot model used in the AI-Native Textbook on Physical AI & Humanoid Robotics.

## Directory Structure

- `body/` - Mesh files for torso and body components
- `head/` - Mesh files for head and facial components
- `limb/` - Mesh files for arms and legs

## File Formats

Mesh files are provided in multiple formats:
- `.stl` - Stereolithography format
- `.dae` - Collada format
- `.obj` - Wavefront OBJ format

## Usage

These meshes are referenced in the URDF files in the `urdf/` directory and can be used for visualization in RViz, Gazebo, and other ROS tools.
